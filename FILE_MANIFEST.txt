FFT Analyzer Network Version - File Manifest
=============================================

This folder contains everything needed to build and run the network version
of the FFT Analyzer on Windows (or Linux).

BUILD INSTRUCTIONS
------------------
Windows (MinGW):
    make -f Makefile.windows

Windows (MSVC):
    nmake /F Makefile.windows CC=cl

Linux:
    gcc fft_analyzer_network.c web_server.c kiss_fft.c -o fft_analyzer_network -lm

QUICK START
-----------
1. Build:    make -f Makefile.windows
2. Test:     ./fft_analyzer_network.exe --test
3. Browser:  http://localhost:8080
4. Network:  ./fft_analyzer_network.exe --source IP:PORT --protocol tcp
